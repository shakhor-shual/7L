# 7L — llama.cpp Model Manager

Web UI для управления моделями llama.cpp. Конструктор параметров, запуск/остановка, просмотр логов, управление контекстом.

## Быстрый старт

```bash
./7L
# Открыть http://localhost:7777
```

## Сборка

```bash
go build -o 7L .
```

## Возможности

- Конструктор аргументов для llama-server
- Добавление/удаление параметров через UI
- Кастомный диалог для --ctx-size
- Запуск и остановка процессов
- Просмотр логов в реальном времени (SSE)
- Управление контекстным окном
- Raw-режим для прямого редактирования аргументов
- Поддержка нескольких конфигураций
